# Heartbeat Checklist

Run through this checklist. Only report items that need attention.

## Always Check
- Are there any failed Vercel deployments? If yes, alert immediately with the error.
- Are there uncommitted changes in any project repo older than 4 hours? If yes, remind me to commit or commit them.
- Is there a content draft in content/drafts/ that's been waiting for review for more than 24 hours? If yes, send a gentle reminder.

## During Work Hours (8 AM - 6 PM)
- Quick scan: any GitHub issues or PRs on my repos that need response?
- Is today's daily log started? If not, create it from the morning plan.
- Have I made at least one commit today? If it's past noon and no commits, flag it.

## Evening (6 PM - 10 PM)
- Has today's daily log been wrapped up with an end-of-day summary?
- Are all project TASKS.md files current?

## Weekly (Monday mornings only)
- Is last week's retrospective written and in content/drafts/?
- Are MEMORY.md and project memory files up to date?

## Maintenance
- If this checklist feels stale or has items that never trigger, update HEARTBEAT.md with a better version.
- If nothing needs attention, reply HEARTBEAT_OK.
