## Project Launched: Watchtower

### What It Is
AI-powered infrastructure monitoring agent that:
- Monitors deployed sites, GitHub repos, APIs, databases
- Detects anomalies (errors, downtime, CI failures)
- Provides AI-generated diagnoses
- Suggests fixes automatically

### Why It's Unique
- Most monitoring tools show data but don't explain it
- Watchtower provides AI diagnoses for every alert
- Proactively watches instead of waiting for queries
